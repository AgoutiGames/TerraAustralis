# TerraAustralis
A demo game for the RawSalmonEngine revolving around a jumping Thylacine.

## Download compile and run
Linux:
    
    $ git clone --recursive https://github.com/AgoutiGames/TerraAustralis.git
    $ cd TerraAustralis
    $ mkdir build
    $ cd build
    $ cmake ../Engine
    $ make
    $ ./Salmon
